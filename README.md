# dictionary-generator

This is an (unfinished) dictionary generator for [Template:Dictionary](https://wiki.teamfortress.com/wiki/Template:Dictionary).

While it does work, translations are not 100% accurate. Use it at your own risk. 

## Running

* Open ``config.js`` and edit it with your settings.
* Install Node.js and run ``node convert.js``.
* Move the public/ directory and its files to a web server.
